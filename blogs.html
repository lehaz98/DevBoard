Question-1: What are the different ways to select an element in the DOM?

1. getElementById() – id দিয়ে একটা নির্দিষ্ট element বাছাই করতে এটা ব্যবহার করা হয়। 2. getElementsByClassName() – একই class-এর সব element বাছাই করতে এটা ব্যবহার করা হয়। এটা একটা HTMLCollection রিটার্ন করে। 3.getElementsByName() – name attribute দিয়ে element বাছাই করতে এটা ব্যবহার করা হয়। 4. querySelector() – CSS selector দিয়ে প্রথম যেই element মিলে সেটাকে বাছাই করতে এটা ব্যবহার হয়।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1 innerHTML. যদি এখানে HTML ট্যাগ দেওয়া হয়, সেটাকে HTML হিসেবে রেন্ডার করে। 2 innerText.কোনো HTML ট্যাগ থাকলে সেটা টেক্সট হিসেবেই ধরে, HTML হিসেবে নয়। 3 textContent.এটা এলিমেন্টের সম্পূর্ণ টেক্সট কনটেন্ট নেয় বা সেট করে, এমনকি যেসব টেক্সট ওয়েবসাইটে দেখা যায় না, সেগুলোকেও।

Question-3: What is event delegation in the DOM?

Event delegation হচ্ছে JavaScript-এর এক ধরণের টেকনিক যেখানে আমরা parent element-এর মাধ্যমে child element-এর ইভেন্ট হ্যান্ডল করি। মানে, অনেকগুলা child element-এর জন্য আলাদা আলাদা ইভেন্ট লিসেনার না দিয়ে, একটাই parent element-এ ইভেন্ট লিসেনার সেট করি।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো DOM-এ ইভেন্টের একধরনের propagation mechanism, যেখানে ইভেন্টটি নিচ থেকে উপরে ওঠে — মানে, যেই element-এ ইভেন্ট ঘটে, সেখান থেকে তার parent, তারপর তার parent… এভাবে পুরো DOM tree-তে উপরের দিকে ছড়িয়ে পড়ে।
